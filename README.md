# Arya Enterprises - E-Seva Kendra Website Template

A modern, responsive, and multi-lingual (English, Marathi, Hindi) frontend template designed for **E-Seva Kendras** and service-based businesses. This project is built using **Tailwind CSS**, **Vanilla JavaScript**, and features a modular SPA-like (Single Page Application) behavior for service rendering.
